Программа отображения рельефа поверхности Марса

Для работы программы необходимо загрузить файлы megR*.img megT*.img с данными рельефа поверхности 
с сайта NASA. Несколько загруженных файлов и настроенный config лежат в папке Result в решении.
разрешение 4 пикселя на градус https://pds-geosciences.wustl.edu/mgs/mgs-m-mola-5-megdr-l3-v1/mgsl_300x/meg004/
разрешение 16 пикселей на градус https://pds-geosciences.wustl.edu/mgs/mgs-m-mola-5-megdr-l3-v1/mgsl_300x/meg016/
разрешение 32 пикселя на градус https://pds-geosciences.wustl.edu/mgs/mgs-m-mola-5-megdr-l3-v1/mgsl_300x/meg032/
разрешение 64 пикселя на градус https://pds-geosciences.wustl.edu/mgs/mgs-m-mola-5-megdr-l3-v1/mgsl_300x/meg064/
Для порядка, для каждого разрешения необходимо создать одельную папку, например, 004 или 032.

Отредактировать файл MarsPlay.config.
Это обычный XML файл, содержащий информацию о расположении файлов, настройках и прочем.
По разделам.
Ведение журнала работы программы:
	<Logger>
		<LogFile>!MarsPlay_log.txt</LogFile>
		<Debug clear="1">On</Debug>
		<Release clear="0">On</Release>
	</Logger>

LogFile -  путь к файлу сообщений.
Debug - тег для работы в отладочном режиме, в окончательном варианте не используется.
Release -  тег для работы в окончательном варианте. 
clear="0" - не стирать предыдущие сообщения.
clear="1" - стирать предыдущие сообщения.
On - вести журнал сообщений.
Off - не вести журнал сообщений.

Угол обзора:
	<ViewAngle>20</ViewAngle> - задание начального угла поля зрения.
	
Данные поверхности по умолчанию.
	<MegdrDefault>1</MegdrDefault>   1 - id данных
	
	<Megdr id="1">
		<RadiusFile>.\004\MEGR90N000CB.IMG</RadiusFile>
		<TopographyFile>.\004\MEGT90N000CB.IMG</TopographyFile>
		<BaseHeight>3396000</BaseHeight>
		<Lines>720</Lines>
		<LineSamples>1440</LineSamples>
	</Megdr>

id - номер для указания данных по умолчанию.
RadiusFile - путь к файлу с данными радиуса
TopographyFile - путь к файлу с данными топологии
BaseHeight - Базовая высота (везде постоянная величина)
Lines и LineSamples разрешение файлов. Информация из вышеуказанных источников из файлов *.lbl. В файле config уже указаны правильные значения для различных разрешений.

Палитра.
	<PaletteDefault>3</PaletteDefault>  3- номер палитры по умолчанию.
	
	<Palette id="1" interpolate="1024">
		<Color Height=" -8199">0x000000</Color>
		<Color Height=" -1000">0x00AA00</Color>
		<Color Height="     0">0x0000FF</Color>
		<Color Height=" 21218">0xFF0000</Color>
	</Palette>

id - номер для указания данных по умолчанию.
interpolate - динамический диапазон значений (разрешение) палитры
Color - тег задания цвета для конктретной высоты.
Height - высота, для которой задается цвет палитры.
0x00AA00 - цвет в формате RGB.


Работа с программой.
Управление движения осуществляется клавишами стрелок, где, вверх и вниз - движения вперед и назад, влево и вправо - соответствующие повороты. 
Мышью движения влево и вправо - аналогично стрелкам, а движение вперед и назад - увеличение и уменьшение высоты камеры.

клавиши 1 и 2 - увеличение и уменьшения коэффициена редьефа. По умолчанию - 5. То есть, гора 1 км, рисуется высотой 5 км.
3 и 4 переключение между палитрами.
5 и 6 переключение между данными с разным разрешением.
- и + изменение скорости движения камеры
Home End изменение угла поля зрения камеры.
Esc - выход из программы.
